### REQ-2: Superpowers 커밋 제안 억제 (컨텍스트 주입)
- **판단:** 가능. no-git-mode에서 이미 동일한 패턴 사용 중
- **가이드라인:** DEVELOP 진입 시 `subagent-driven-development` 호출에 "태스크 완료 후 커밋 제안 금지, 완료 보고만" 규칙을 컨텍스트로 전달
- **리스크:** 중간 — Superpowers가 컨텍스트를 무시할 가능성 존재 (텍스트 기반 지시이므로 100% 보장 불가)
- **결정:** 리스크 수용, 실 사용 후 필요 시 추가 대응
