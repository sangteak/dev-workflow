## 3. 재논의 기록

- **이슈 카드 관점 줄 적용 범위 축소안** (🎯 WFD 제안: "설계 서술을 바꾸는 이슈만") — 사용자가 재논의 선택. 재논의 라운드에서 🛠️ CCE [Contrarian]가 반박: ① 실질성 규칙과 이중 통제(중복 규범) ② 조건 판정 불신의 해법이 또 하나의 조건 판정인 자기모순 ③ 실패 방향 비대칭 — 축소안의 최악은 "경미해 보였는데 기술 결정을 건드린 이슈"에서 관점이 침묵(그 뒤 확인 지점 없음), 현행안의 최악은 소음 몇 줄. 대안으로 기본값 반전 제시. 🎯 WFD 철회·합류 → 결정 14로 확정.
